# Apresentação do Projeto:

Projeto destinado à avaliação acadêmica da Universidade Federal Rural de Pernambuco (Campus Dois Irmãos – Sede), do curso de Bacharelado em Sistemas de Informação (DEINFO), na disciplina Projeto Interdisciplinar para Sistemas de Informação II, ministrada pelo docente Cícero Garrozzi. O projeto foi elaborado pelos discentes Bruno Lins, Cayo Menezes, Guilherme Abraão e Thyago Murilo.
